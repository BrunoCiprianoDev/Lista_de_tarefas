-> Descrição:
	O projeto "Lista de Tarefas" é um aplicativo de produtividade que permite a organização de tarefas definidas por titulo 
e data a partir de uma interface simples e intuitiva;

-> Objetivos:
	Além do objetivo de criar uma aplicação útil capaz de facilitar a organização das minhas tarefas diarias como estudante
de Sistemas de Informaçao, o projeto em questão também tem foco a pragmatização dos meus conhecimentos em Java e sua integração com 
o sistema de gerenciamento de banco de dados MYSQL através da JDBC(Java Database Connectivity).

-> Como executar a aplicação:
1)Crie um banco de dados Mysql nomeado por lista_tarefas;
2)Carregue a tabela lista_tarefas.sql(contida na pasta DUMP) no banco de dados nomeado por lista_tarefas;
3)Copie o arquivo ListaDeTarefas.jar em seu diretorio de arquivos;
4)A partir de um terminal (Linux/Windows) execute o comando java -jar ListaDeTarefas;
5)Na tela de login inicial, informe o usuario e senha de seu sistema de gerenciamento de banco de dados;
